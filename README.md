# Disease Mapping Project

This repository contains rules and workflows for mapping pharmaceutical indication data to:

1. ICD-10 disease classification
2. Internal disease taxonomy

## Files

- `DISEASE_MAPPING_RULES.md`
  - Detailed mapping rules and execution instructions for Codex

## Workflow

Raw pharmaceutical indication data
→ Disease entity extraction
→ ICD-10 mapping
→ Internal disease classification mapping
→ Unmapped record review
